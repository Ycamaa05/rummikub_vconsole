Projet : Jeu Rummikub
Langage : C
Version actuelle : Prototype console

Description :
Ce projet est un prototype du jeu Rummikub développé en langage C.
Il permet de jouer une partie complète en mode console, avec gestion
des joueurs, des tuiles, du plateau et des scores.

Compilation :
gcc *.c -o rummikub_console

Exécution :
wsl ./rummikub_console

Remarque :
L’interface graphique et l’IA seront développées ultérieurement.
